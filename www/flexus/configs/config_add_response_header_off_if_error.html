add-response-header-off-if-error

add-response-header-off-if-error : "true or false";

add-response-header-off-if-error 설정은 응답 헤더를 추가한다. “name : value” 형태로 응답 헤더를 추가할 수 있으며 다음 사항을 유념한다.

  1. 헤더 이름은 문자열이며HTTP/1.1 프로토콜에 정의한 유효한 문자만을 사용한다.

  2. 헤더 값은 문자열이며 HTTP/1.1 프로토콜에 정의한 유효한 문자만을 사용한다.

참고

add-response-header 설정은 이름에서 알 수 있듯이 헤더를 추가만 하기 때문에 동일한 이름의 응답 헤더를 여러개 추가할 수 있다. 동일한 이름의 응답 헤더를 삭제하거나 변경하지 않기 때문에 사용에 주의해야 한다.

http["80"] : {
    add-response-header : {
        "a" : "b";
        "b" : "c";
    }
}
